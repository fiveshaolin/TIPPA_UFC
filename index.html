Scoring

How leaderboard scoring works

Latest event rows show the event total first, followed by the scoring breakdown in brackets.

14 (5+2+3+0+4)

Base score only, no extra bonuses.

16 (5+2+3+0+4) +(2)

Base score plus highest score bonus.

26 (5+2+3+0+4) +(2+10)

Base score plus highest score bonus and flush bonus.

36 (5+2+3+0+4) +(2+10+10)

Base score plus highest score bonus, flush bonus, and royal flush bonus.

Base score The values inside the first brackets, for example (5+2+3+0+4).
Highest score bonus A bonus added when a player has the top score for the event.
Flush bonus An extra bonus added when the flush condition is achieved.
Royal flush bonus An extra bonus added on top when the royal flush condition is achieved.
